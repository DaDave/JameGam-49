# JameGam-49 - Design Document

## Introduction

### Game Summary

The game is a 2D adventure in which the protagonist plays an intern at the Explocorp explosives factory. Due to an unfortunate accident, the company's self-destruct mechanism has been triggered. Now it's time to run for your life!

### Inspiration

**Inspiration A**

Among us

**Inspiration B**

Bullet Hell

### Player Experience

The player controls the protagonist through 2D scenes and tries to find a way to save himself by hiding inside a explosion-proof box. Everything can explode, better be careful.

### Platform

The game is developed to be playable in web.

### Development Software

 * Godot as game engine
 * MediBand Paint pro for graphics design
 * Paint.NET for graphics design

### Genre

- Adventure
- Singleplayer

### Target Audience

The game is designed for people who like games with hurry but also exciting adventure experience where the protagonist has to escape an exploding factory as soon as possible.

## Concept

### Gameplay overview

The player navigates the protagonis through the scene by using arrow keys on the map. With the dodge ability, the player evades projectiles which will be thrown after exploding interiors.

### Theme Interpretation (Explode)

Every objective in the room could possible explode. The character has to be careful because entering a special radius of an object can trigger a explosion countdown.

### Special Object Interpretation (Box)

The box is players safe space. By finding a way to open the box the player is able to hide inside the box.

### Mechanics

| Mechanic                | Description                                                                                                         |
|-------------------------|---------------------------------------------------------------------------------------------------------------------|
| Movement                | By pressing W,A,S,D (or arrow keys) the player moves the the protagonist through the factory.                       |
| Dodge                   | By pressing SPACEBAR the player is able to evade projectives after explosions.                                      |
